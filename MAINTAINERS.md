# Maintainers

This file documents ownership and release responsibilities for OSS.

## Primary Maintainer

- GitHub: `@haxxornulled`
- Responsibilities:
  - architecture and roadmap
  - release approval
  - NuGet.org publish credential ownership

## Backup Maintainer Requirement

Current state: `Unfilled`

Target state:

- at least one additional maintainer with:
  - repo write access
  - NuGet publish access
  - runbook familiarity (`docs/RELEASE_RUNBOOK.md`)

## Operational Rule

No "Production Default" designation until backup maintainer role is filled.

## Escalation

If primary maintainer is unavailable:

1. Freeze feature merges.
2. Allow only reliability/security fixes.
3. Use documented runbook for emergency release operations.
